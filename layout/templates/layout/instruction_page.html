{% blocktrans %} Parent's time je platforma pomoću koje možete stupiti u kontakt sa bebisiterima/dadiljama koje pružaju usluge čuvanja dece i ukoliko odgovarate jedni drugima možete ih vrlo brzo i jednostavno kontaktirati i početi sa radom. {% endblocktrans %}

{% trans "Platforma funkcioniše na sledeći način:" %}

{% trans "Registracija korisnika" %} {% blocktrans %} - ukoliko želite da pristupite bazi dostupnih bebisitera/dadilja morate biti registrovani korisnik platforme parentstime.rs. Registrujete se pomoću email adrese i lozinke. Obratite pažnju na to da lozinka bude što bolja kako biste što bolje zaštitili svoje podatke. Nakon registracije potrebno je da verifikujete svoju email adresu odlaskom na vaš email nalog i klikom na verifikacioni link. {% endblocktrans %}

{% trans "Popunjavanje profila" %} {% blocktrans %} - nakon uspešne registracije i verifikacije, prelazite na sledeći korak. Naredni korak je popunjavanje informacija na vašem profilu. Kreiranje profila se sastoji iz dva koraka: popunjavanja osnovnih informacija o sebi i dodavanja slike. Vaš profil je dostupan samo bebisiterima/dadiljama kojima vi pošaljete zahtev za rezervaciju. {% endblocktrans %}

{% trans "Ažuriranje vremena" %} {% blocktrans %} - u ovom koraku je potrebno da dodate informacije o vremenu kada vam je potrebna usluga čuvanja dece. Možete izabrati dane kao i doba dana kada vam je usluga potrebna (ujutru, popodne, uveče ili noću). {% endblocktrans %}

{% blocktrans %} Kreiranje profila je završeno. Čestitamo. Sada možete pristupiti bazi dostupnih bebisitera/dadilja i stupiti u kontakt sa njima. {% endblocktrans %}

{% trans "Ažuriranje profila -" %} {% blocktrans %} u svakom trenutku možete da ažurirate vaš profil i vreme kada vam je potrebno čuvanje dece. {% endblocktrans %}

{% trans "Baza bebisitera/dadilja" %} {% blocktrans %} - Sada kada ste registrovani korisnik možete pristupiti bazi bebisitera/dadilja i vrlo brzo i jednostavno pronaći osobu koja odgovara baš vašim kriterijumima pomoću {% endblocktrans %}

{% trans "Poklapanje sa bebisiterima/dadiljama" %} {% blocktrans %} - na stranici baza bebisitera/dadilja, koja će vam biti dostupna nakon registracije, možete videti sve dostupne bebisitere/dadilje i filtrirati ih baš prema vašim kriterijumima. Nakon što pronađete odgovarajućeg bebisitera/dadilju možete videti njihov profil sa detaljnim informacijama i rezervisati korisnika. {% endblocktrans %}

{% trans "Rezervacija" %} {% blocktrans %} - Kada pošaljete zahtev za rezervaciju bebisiter/dadilja ga može prihvatiti ili odbiti. Nakon što je vaša rezervacija prihvaćena dobijate mogućnost ocenjivanja, komentarisanja i prijavljivanja korisnika. Vaš profil takođe u ovom slučaju može biti ocenjen, komentarisan i prijavljen od strane bebisitera/dadilja. {% endblocktrans %}
